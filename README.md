# EDA Projects Folder

This folder contains multiple Exploratory Data Analysis (EDA) projects on different datasets.

Each subfolder represents a separate project with its own data, notebooks, and documentation.

All analyses are done for educational purposes only.

**Important:**  
The datasets used in these projects are owned by their original creators and providers.  
Ownership and rights remain with them. Proper credit is given in each project’s documentation.

Please refer to individual project folders for dataset sources, licenses, and detailed information.
